**水平居中**

1. 元素为行内元素，

```js
.inner {
    display: inline/inline-block;
    text-align: center;
}
```

2. 块级元素，宽度固定

```js
margin: 0 auto
```

**垂直居中**

1. flex布局

```
display:flex
justify-content:center; //实现水平居中
align-items:center    //实现垂直居中
```

2. gird布局

```
display: grid;
justify-items: center;
align-items: center;
```
